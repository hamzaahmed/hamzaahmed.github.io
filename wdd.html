Web Design DeCal
Teaching web design to UC Berkeley students
Web Design DeCal, affectionately nicknamed "WDD", is the most popular student-taught class at UC Berkeley. We offer the most beginner-friendly entry point into web development, providing support along the way. I joined as an instructor for WDD in Fall 2014, when we received a whopping 905 applicants for the course, where we could only accept 100. My vision is that we would be the standard for learning web design for anyone with an internet connection.
