Assignment

Note: Please review the lecture slides before you go through this assignment.

In this assignment, You will be creating a website about yourself. You will use 100% pure HTML without CSS, so it probably won't look that great, but will still contain structured information! (That's the purpose of HTML anyways.)

You will need to create a workspace folder with a name of your choice for this assignment. Remember that for each website you create, you need a workspace folder with index.html, style.css, and (maybe) script.js (review the lecture slides to structure your workspace folder correctly.) You will continue to improve this homework's website in next week's homework and onwards.

Here are the requirements for the website:

  1. Write your name in huge, bold text (Hint: I'm MAD)
  2. On the next line, write your major and year in italic text (Hint: I'm fancy)
  3. Add a paragraph about yourself
  4. Add a picture of yourself so we can recognize you in class! (where should your pictures go in the workspace folder?)
  5. Add a link that, when clicked, directs you to the most beautiful website you have ever seen. The text within the link should be the name of the website.
  6. Write your favorite song in text that is both bold and italic. If we like the song you pick, we might include it in one of our class playlists!
